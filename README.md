## ローカルで実行する
Hugoをインストール
```powershell
winget install Hugo.Hugo.Extended
```
githubのリポジトリをクローン
```powershell
git clone https://github.com/ki1989/hugo-blog.git
```
Hugoを起動
```powershell
hugo server
```
